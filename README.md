# OracleRead

OracleRead is a free and open source community manga reader for Android 6.0 and above.

OracleRead is a community-driven fork of Mihon focused on providing a clean, stable, and modern manga reading experience.

Project lineage:

Tachiyomi -> Mihon -> OracleRead

## First Release Goal

OracleRead v1.0.0 is intended to be a polished Mihon fork with OracleRead branding, new iconography, a new Android package name, Keiyoushi extension repository compatibility, proper attribution, and no functional regressions.

Feature additions will be implemented in future versions.

## Extension Repository

OracleRead includes the Keiyoushi and Yūzōnō extension repositories by default:
